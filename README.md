# Spam Detector using Python

## Objective
Build a machine learning model to classify messages as Spam or Not Spam.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Naive Bayes
- TF-IDF Vectorization

## Steps
1. Load and clean dataset
2. Convert text into numerical form
3. Train Naive Bayes model
4. Evaluate accuracy
5. Predict new messages

```bash
pip install -r requirements.txt
python spam_detector.py
